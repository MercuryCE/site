Mercury is a simple but powerful package manager for Halo Custom Edition.

It allows you to download and install any mod, map or addon from a command line terminal.

mercury install chimera Searching for "chimera" in repository... done. 1.19MB received, 2.47MB/s throughput, 0s elapsed Inserting chimera files... Warning, creating backup for conflict file: "strings.dll"... done. Success, package "chimera" has been installed.
Downloads Search packages Join us on Discord

NOTE:Mercury only works for Halo Custom Edition on Windows and Linux.
